What is the correct format of a User Story?

"As a [actor], I want to [goal] so that [benefit]."
Example: As a frequent traveler, I want to access my travel schedule offline so that I don't have to worry about internet connectivity while I'm out of town.

How does it work?

Visit the “Generate” website
Create an account by entering your name, email address, mobile number and password.
If the account has been successfully created, you can login to start generating
On the “home” page, we can create new projects, can search for projects that we have created, and there is a project list feature to see what our projects are.
To generate User Story to User Story Scenario, we can fill in the project title, project description, and user story text that we want to generate.
If we have filled in these three things, we can click generate to generate a User Story Scenario.
The results will appear on a new page
We can click “See Project List to see our project
There, we can see information about the project, such as the project name, the date the project was created, the last date the project was edited, and perform several actions such as generate, edit, and delete.
To generate Activity Diagram or Sequence Diagram, we can choose the orange generate icon.
A new page will appear to choose what diagram we want to generate.
If you have chosen, the results of the generate diagram will appear on a new page
